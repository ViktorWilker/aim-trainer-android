# Aim Trainer

A mobile aim trainer game for Android built with Kotlin and Jetpack Compose.

## Gameplay
Targets appear randomly on a grid. Tap them as fast as you can before the time runs out.

## Features
- 3x3, 4x4 and 5x5 grid modes
- Configurable timer (15s, 30s, 60s)
- Combo multiplier

## Built with
- Kotlin
- Jetpack Compose
- Navigation Component
